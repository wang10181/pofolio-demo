Project Title One — Photography Collection

This project showcases a curated photography series exploring visual storytelling through composition, lighting, and contextual design. The full collection can be viewed here:

View Live Project:
https://wang1018.myportfolio.com

This photo series contains 9–12 images, each intentionally crafted to highlight a different visual strategy. The idea behind the shoot centers on capturing everyday moments from a fresh perspective—transforming ordinary scenes into expressive visual narratives.

The collection demonstrates techniques such as:
- Compositional contrast (rule of thirds, symmetry, leading lines)
- Exposure & tonal control (natural light, soft shadows)
- Background design (minimalism, environmental storytelling)

Photographic styles explored:
- Macro details
- Still-life
- Architecture
- Candid moments
- Portrait session introducing character and mood

Overall, this project highlights my ability to compose visually engaging imagery that communicates emotion and narrative through professional photographic principles.

Project Title Two — Bootstrap Portfolio Website

This project is a fully responsive, multi-page portfolio website built using the Bootstrap framework, demonstrating clean layout, semantic HTML, accessibility, and responsive design. The full site can be viewed here:

View Live Project:
https://wang10181.github.io/mtm6201-final/

The website contains three pages (Home, About/Work, Contact), each structured with semantic HTML elements such as <header>, <main>, <section>, and <footer>.

Techniques and design considerations include:
- Responsive layout using Bootstrap grid system
- Optimized images with <picture>, srcset, and sizes attributes for different screen sizes
- Accessibility enhancements: alt text, skip links, ARIA roles, keyboard navigation
- Clean file/folder structure with proper naming conventions

Features explored:
- Multi-page navigation
- Mobile-first design and adaptive layouts
- Semantic HTML structure
- Accessibility and usability best practices

Overall, this project demonstrates my ability to build a professional, responsive, and accessible portfolio website using modern front-end techniques and frameworks.

View Project Repository:

The application is structured using a single HTML file (index.html), a single CSS file (style.css), and a single JavaScript file (script.js). It follows modern development practices:

Techniques and features include:

Development workflow and report:
- Mockup, static prototype, and final product created in stages
